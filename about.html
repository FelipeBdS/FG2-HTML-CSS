FG2 Multimarcas

Escolhemos o tema de loja virtual para um projeto acadêmico de desenvolvimento web e decidimos fazer uma loja fictícia de camisas multimarcas. Temos 8 marcas disponíveis com 3 camisas cada (optamos por ter poucos produtos por questões de complexidade no banco de dados visto que também temos que ficar atentos ao prazo).
